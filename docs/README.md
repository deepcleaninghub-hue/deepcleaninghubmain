# Deep Cleaning Hub - Public Pages

This directory contains public-facing web pages hosted via GitHub Pages.

## Pages

- **index.html** - Landing page
- **privacy-policy.html** - Privacy Policy (required for App Store & Play Store)

## Live URLs (after GitHub Pages is enabled)

- Landing: `https://deepcleaninghub-hue.github.io/deepcleaninghubmain/`
- Privacy Policy: `https://deepcleaninghub-hue.github.io/deepcleaninghubmain/privacy-policy.html`

## Setup GitHub Pages

1. Go to repository settings: https://github.com/deepcleaninghub-hue/deepcleaninghubmain/settings/pages
2. Under "Source", select "Deploy from a branch"
3. Select branch: `main`
4. Select folder: `/docs`
5. Click "Save"
6. Wait 1-2 minutes for deployment

## Usage

Use the privacy policy URL when submitting to:
- Google Play Console
- Apple App Store Connect
